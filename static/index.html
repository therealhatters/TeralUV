Teral UV

Python-hosted Ultraviolet-style frontend

UV-style setup, without npm.

This frontend runs against a Python static server plus a Wisp backend. It loads the Ultraviolet stack, configures a transport, and opens proxied pages inside the viewer below.

When the Python UV backend is running and the service worker is registered, pages will load through the Ultraviolet prefix inside this viewer.
